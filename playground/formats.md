# All Supported Color Formats

Hover over any swatch to see format conversions and WCAG contrast info. Click to copy.

## Hex

| Format | Example |
|--------|---------|
| 3-digit | `#f60` |
| 6-digit | `#ff6600` |
| 8-digit (alpha) | `#ff660080` |
| 4-digit (alpha) | `#f608` |

## RGB

| Format | Example |
|--------|---------|
| Classic | `rgb(255, 100, 0)` |
| With alpha | `rgba(255, 100, 0, 0.5)` |
| Modern (space) | `rgb(255 100 0)` |
| Modern (alpha) | `rgb(255 100 0 / 50%)` |

## HSL

| Format | Example |
|--------|---------|
| Classic | `hsl(30, 100%, 50%)` |
| With alpha | `hsla(30, 100%, 50%, 0.5)` |
| Modern (space) | `hsl(30 100% 50%)` |
| Modern (alpha) | `hsl(30 100% 50% / 50%)` |

## OKLCH / OKLAB

| Format | Example |
|--------|---------|
| OKLCH | `oklch(70% 0.15 30)` |
| OKLCH (alpha) | `oklch(70% 0.15 30 / 80%)` |
| OKLAB | `oklab(70% 0.1 -0.05)` |

## WCAG Contrast Test

These colors should show different contrast levels:

- High contrast vs white: `#000000` `#1e293b` `#171717`
- Low contrast vs white: `#f0f0f0` `#e5e5e5` `#fef9c3`
- High contrast vs black: `#ffffff` `#fef08a` `#e0f2fe`
- Mid contrast: `#6b7280` `#94a3b8` `#64748b`
